# multimedia.ahk
Simple ahk made for small keyboard to emulate multimedia keys.


<kbd>Left Control</kbd> + <kbd>Left Alt</kbd> + <kbd>-</kbd> - Decrease volume
                                                <kbd>+</kbd> - Increase voulume
                                                <kbd>\</kbd - Mute
                                                <kbd>Arrow left</kbd> - Previous
                                                <kbd>Arrow right</kbd> - Next
